# 🤖 AI Systems Lab

Welcome to the **AI Systems Lab**! This repository serves as a centralized hub for experimenting with Large Language Models (LLMs), Retrieval Augmented Generation (RAG), and autonomous agents.

## 📁 Project Structure

This repository is organized as a **monorepo**. Each subdirectory is a standalone project with its own dependencies to ensure isolation:

- `chatbot-cli/` 💬 — CLI-based AI interactions and prompt engineering.
- `rag-experiments/` 📚 — Vector database implementations and retrieval testing.
- `agent-lab/` 🛠️ — Autonomous agent workflows and tool-use exploration.
- `vector-db-lab/` 🔍 — Benchmarking different vector store performances.
- `notes/` 📝 — Learning logs, research papers, and documentation.
